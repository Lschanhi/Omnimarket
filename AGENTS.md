# Instrucoes do Projeto

- Ao criar novas branches neste projeto, use nomes em portugues.
- Nao use o prefixo `codex/` nas branches deste projeto.
- Use `kebab-case`, sem espacos e sem acentos.
- Prefira nomes curtos e descritivos do objetivo.
- Exemplos de nomes esperados: `ajuste-deploy-render`, `refino-home-lojas`, `correcao-fluxo-pagamento`.
- Se o usuario informar um nome de branch especifico, esse nome tem prioridade.
